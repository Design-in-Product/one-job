# Keyboard Shortcuts

One Job is fully keyboard accessible. You can navigate and interact with the entire application using only your keyboard.

## Global Shortcuts

| Key | Action |
|-----|--------|
| <kbd>M</kbd> | Open menu |
| <kbd>Esc</kbd> | Close menu / Close modal |

## Card Deck View

When viewing your task stack:

| Key | Action |
|-----|--------|
| <kbd>Enter</kbd> or <kbd>Space</kbd> | Flip card to reveal task details |
| <kbd>Shift</kbd> + <kbd>→</kbd> | Complete current task (move to completed) |
| <kbd>Shift</kbd> + <kbd>←</kbd> | Defer current task (move to bottom of stack) |
| <kbd>Tab</kbd> | Navigate to task details (when card is flipped) |

## Menu Navigation

When the menu is open:

| Key | Action |
|-----|--------|
| <kbd>↑</kbd> / <kbd>↓</kbd> | Navigate between menu items |
| <kbd>←</kbd> / <kbd>→</kbd> | Navigate between menu items |
| <kbd>Tab</kbd> | Cycle through menu items (forward) |
| <kbd>Shift</kbd> + <kbd>Tab</kbd> | Cycle through menu items (backward) |
| <kbd>Enter</kbd> or <kbd>Space</kbd> | Select focused menu item |
| <kbd>Esc</kbd> | Close menu |

## Task Details Modal

When viewing task details:

| Key | Action |
|-----|--------|
| <kbd>Tab</kbd> | Navigate through form fields and buttons |
| <kbd>Shift</kbd> + <kbd>Tab</kbd> | Navigate backwards |
| <kbd>Esc</kbd> | Close modal |
| <kbd>Enter</kbd> | Submit form (when focused on Save button) |

## Forms

### Task Creation Form

| Key | Action |
|-----|--------|
| <kbd>Tab</kbd> | Move to next field |
| <kbd>Shift</kbd> + <kbd>Tab</kbd> | Move to previous field |
| <kbd>Enter</kbd> | Submit form (when focused on Add Task button) |
| <kbd>Esc</kbd> | Cancel and close form |

### Task Editing

| Key | Action |
|-----|--------|
| <kbd>Tab</kbd> | Navigate between title, description, and buttons |
| <kbd>Enter</kbd> | Save changes (when focused on Save button) |
| <kbd>Esc</kbd> | Cancel and close without saving |

## Navigation Views

### Completed Tasks View

| Key | Action |
|-----|--------|
| <kbd>Tab</kbd> | Navigate through completed tasks list |
| <kbd>Enter</kbd> | View task details |
| <kbd>Esc</kbd> or Back button | Return to main task stack |

### Integrations View

| Key | Action |
|-----|--------|
| <kbd>Tab</kbd> | Navigate through integration options |
| <kbd>Enter</kbd> | Select/configure integration |
| <kbd>Esc</kbd> or Back button | Return to main task stack |

## Accessibility Features

### Screen Reader Support

One Job is optimized for screen readers with:

- **Live regions** that announce task counts, loading states, and errors
- **Descriptive ARIA labels** on all interactive elements
- **Status announcements** when menus open or actions complete
- **Semantic HTML** with proper heading hierarchy
- **Hidden decorative elements** that don't add meaningful information

### Keyboard Hints

- Visual hints appear on screen showing available keyboard shortcuts
- Menu overlay shows: "Arrow keys to navigate • Enter to select • Esc to close"
- Card deck shows: "Press Enter or Space to flip"

### Focus Management

- Clear **focus rings** on all interactive elements
- **Logical tab order** following visual layout
- **Focus trap** in modals (Tab stays within modal)
- **Auto-focus** on first element when menus open
- **Focus return** to triggering element when closing modals

## Tips for Keyboard Users

1. **Start with the Card Deck**: Press <kbd>Tab</kbd> until you reach the main task card, then use <kbd>Enter</kbd> to flip it

2. **Quick Actions**: Use <kbd>Shift</kbd> + <kbd>→</kbd> / <kbd>←</kbd> for fast task management without opening details

3. **Menu Access**: Press <kbd>M</kbd> from anywhere to open the menu

4. **Form Navigation**: In forms, <kbd>Enter</kbd> submits when focused on the submit button, but adds line breaks in description fields

5. **Cancel Actions**: <kbd>Esc</kbd> key always closes the current modal or menu
