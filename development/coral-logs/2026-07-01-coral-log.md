## Root causes worth cross-pollinating

- **Vite `host: "::"` is IPv6-only** — the cause of the August-2025 "dev
  server unreachable" mystery. Use `host: true`.
- **`max-w-md mx-auto` in a flex column has zero intrinsic width** once all
  children are w-full; a fixed-width child had been silently propping the
  layout open.
- **Tailwind keyframes referenced only from inline styles never ship** —
  the utility class must appear in markup for emission.
- **Playwright screenshots during animations** beat reasoning about gesture
  code; drove all verification at 390x844.
